# منصة إدارة الفيديوهات والصور والمحتوى الإلكتروني (التحكُم، التعديل، التوزيع)

## ملخص المشروع

منصة إدارة الفيديوهات والصور والمحتوى الإلكتروني هي نظام متكامل يتيح للمستخدمين إدارة وتحكُم وتعديل المحتوى الإلكتروني بسهولة وسرعة. يمكن للمستخدمين إضافة وتصفح وتعديل الفيديوهات والصور والمحتوى الإلكتروني، كما يمكنهم توزيع المحتوى عبر مختلف المنصات.

## هيكل المشروع

* `backend/` : الملفات الخاصة بالباكند
 + `app.py` : الملف الرئيسي للباكند
 + `models.py` : definitions of database tables
 + `routes.py` : definitions of API routes
 + `schemas.py` : definitions of API request and response schemas
* `frontend/` : الملفات الخاصة بالفرونتند
 + `index.html` : الصفحة الرئيسية للفرونتند
 + `styles.css` : ملفات CSS الخاصة بالفرونتند
 + `script.js` : ملفات JavaScript الخاصة بالفرونتند
* `docker-compose.yml` : ملف تكوين docker-compose
* `requirements.txt` : قائمة المتطلبات الخاصة بالمشروع

## إعداد البيئة باستخدام docker-compose

1. تأكد من أن docker-compose مثبت على جهازك.
2. افتح المجلد الرئيسي للمشروع في terminal.
3. استخدم الأمر التالي لتشغيل البيئة باستخدام docker-compose:
bash
docker-compose up

4. افتح متصفحك واكتب `http://localhost:8000` في العنوان.
5. سوف تظهر لك الصفحة الرئيسية للفرونتند.

## وصف الموديولات والجداول والدوائر

### الموديولات

* `video` : الموديول الخاص بالفيديوهات
* `image` : الموديول الخاص بالصور
* `content` : الموديول الخاص بالمحتوى الإلكتروني

### الجداول

* `users` : جدول المستخدمين
* `videos` : جدول الفيديوهات
* `images` : جدول الصور
* `contents` : جدول المحتوى الإلكتروني

### الدوائر

* `admin` : الدائرة الخاصة بالمسؤولين
* `moderator` : الدائرة الخاصة بالمحررين
* `user` : الدائرة الخاصة بالمستخدمين
